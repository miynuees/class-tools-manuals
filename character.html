칭찬 캐릭터 키우기

학급 칭찬으로 함께 키우는 우리만의 캐릭터 🌱

https://miynuees.github.io/class-character-share/

🌟 어떤 도구인가요?

우리 반이 함께 칭찬을 모아서 캐릭터를 키우는 학급 운영 게임이에요. 학생 개개인이 아니라 반 전체가 한 마음으로 한 캐릭터를 키워나가요.

알에서 시작해서 칭찬을 받을 때마다 포인트가 쌓이고, 5단계까지 성장해요. 한 캐릭터를 다 키우면 명예의 전당에 보관되고, 새 알을 다시 골라서 키울 수 있어요. 총 4종류의 캐릭터를 모두 모으는 게 목표예요!

💡 추천 사용 환경: 교실 전자칠판 (가로 와이드 화면 + 터치 입력에 최적화되어 있어요)

🚀 시작하는 방법

  1. 전자칠판 브라우저(크롬/엣지)로 사이트 열기 위에 있는 주소를 주소창에 입력하세요. 즐겨찾기에 추가해두면 편해요!
  2. 알 4개 중 하나를 골라서 이름 지어주기 알 안에 어떤 캐릭터가 있을지는 비밀! 색깔만 보고 골라요. (예: '무럭이', '복덩이' 등 학급 분위기에 어울리는 이름)
  3. 학급 활동 중 칭찬할 일이 생기면 칭찬 버튼 누르기 오른쪽 칭찬 폼에서 ① 칭찬 종류 선택 → ② 포인트 선택 → ③ 선생님 서명 → ④ '칭찬 주기' 버튼
  4. 레벨업과 완성을 함께 축하하기 레벨이 오를 때마다 변신해요! 100포인트를 채우면 완전체가 되고 명예의 전당에 보관돼요.
  5. 새 알을 골라서 다음 캐릭터 키우기 4종을 모두 모으면 컬렉션 완성! (드래곤, 여우, 유니콘, 펭귄)

🖥️ 전자칠판에 앱으로 설치하기

매번 주소를 입력하지 않아도, 바탕화면 아이콘 하나로 앱처럼 바로 열 수 있어요!

방법 1. 크롬(Chrome)에서 설치하기 ⭐ 추천

  1. 크롬으로 사이트 접속 주소창에 위 사이트 주소를 입력해서 들어가세요.
  2. 주소창 오른쪽 끝의 설치 아이콘(💻 또는 ⊕) 클릭 주소창 안쪽 오른쪽 끝에 작은 설치 아이콘이 떠 있어요. 안 보이면 메뉴(⋮) → '캐스트, 저장 및 공유' → '페이지를 앱으로 설치' 를 눌러주세요. (크롬 버전에 따라 메뉴 위치가 살짝 달라요.)
  3. 팝업이 뜨면 '설치' 클릭 바탕화면과 시작 메뉴에 앱 아이콘이 자동으로 생겨요!
  4. 앱 아이콘 더블클릭으로 실행 주소창·탭 없이 앱처럼 열려서 전자칠판 화면을 더 넓게 쓸 수 있고, 실수로 다른 페이지로 이동할 일도 없어요.

방법 2. 엣지(Edge)에서 설치하기

  1. 엣지로 사이트 접속 주소창에 사이트 주소를 입력해서 들어가세요.
  2. 오른쪽 위 메뉴(⋯) → '앱' → '이 사이트를 앱으로 설치' 클릭 팝업이 뜨면 '설치'를 눌러주세요. 바탕화면에 앱 아이콘이 자동으로 생기고, 크롬과 똑같이 앱처럼 열려요.
💡 크롬을 추천해요! 대부분의 선생님들이 쓰시는 브라우저라 가장 편하고, 한 번 설치해두면 매일 클릭 한 번으로 바로 열 수 있어요. 엣지를 쓰셔도 동일하게 앱 설치가 가능해요.

💾 데이터 저장에 대해

이 도구는 각 전자칠판 안에만 데이터가 저장돼요. 인터넷 서버에 올라가지 않아서 개인정보 걱정이 없고, 학급별로 완전히 독립적으로 운영돼요.

⚠️ 꼭 알아두실 점

  • 같은 컴퓨터에서 브라우저를 바꾸면 데이터가 따로 저장돼요. 항상 같은 브라우저(예: 크롬)로 접속해 주세요.
  • 시크릿/InPrivate 모드는 사용 금지! 창을 닫으면 모든 데이터가 사라져요.
  • 브라우저 '사이트 데이터 삭제'를 누르면 칭찬 기록이 모두 날아가요. 청소 작업할 때 주의하세요.
  • 안전을 위해 정기적으로 백업을 권장해요. (⚙️ 설정 → 데이터 내보내기)
💾 백업하는 방법: 헤더 우측 ⚙️ 버튼 → '데이터 내보내기' 클릭 → JSON 파일이 다운로드돼요. 학기 말이나 중요한 시점에 한 번씩 내보내서 USB나 클라우드에 보관해두세요. 복원도 같은 메뉴에서 '데이터 가져오기'로 가능해요.

❓ 자주 묻는 질문

다른 학교나 다른 반의 데이터와 섞이지 않나요?
절대 섞이지 않아요. 데이터는 각 컴퓨터 브라우저 안에만 저장되어서, 우리 반 전자칠판은 우리 반 데이터만 가지고 있어요.
집에서 미리 칭찬을 기록해둘 수 있나요?
가능하지만, 집 컴퓨터와 학교 전자칠판은 별개로 작동해요. 합치고 싶다면 '데이터 내보내기'로 백업한 후 다른 곳에서 '데이터 가져오기'를 해야 해요.
실수로 칭찬을 기록했어요. 취소할 수 있나요?
개별 칭찬 삭제 기능은 없어요. 큰 실수라면 백업해둔 이전 데이터를 복원하는 방법이 있어요.
캐릭터를 완성하기 전에 다른 알로 바꿀 수 있나요?
현재 캐릭터를 키우는 중에는 자동으로 바뀌지 않아요. 완성한 후 다음 알을 고를 때 새 캐릭터를 시작할 수 있어요.
학급에 따라 칭찬 종류를 바꿀 수 있나요?
현재는 6가지로 고정되어 있지만, '직접 입력' 칸에 원하는 칭찬 내용을 자유롭게 적어서 사용할 수 있어요.
스마트폰으로도 사용 가능한가요?
가능하지만, 가로로 넓은 화면(전자칠판/태블릿)에 최적화되어 있어요. 학급 활동용이라 큰 화면을 권장해요.
