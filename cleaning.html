청소 역할 뽑기

두근두근 카드 뽑기로 청소 시간이 즐거워져요 🧹✨

https://miynuees.github.io/cleaning-role-picker/

🌟 어떤 도구인가요?

청소 시간에 학생들이 직접 카드를 뽑아 자기 역할을 정하는 학급 운영 도구예요. 매번 "오늘은 누가 뭐 할래?" 하지 않아도, 알록달록 카드를 한 번 탭하면 짜잔! 자기 역할이 나와요.

역할은 우리 반 청소 환경에 맞게 자유롭게 추가·수정할 수 있고, 청소 중에는 끝낸 사람이 자기 카드를 탭해서 진행률을 함께 확인할 수 있어요. 다 끝나면 화려한 폭죽으로 마무리까지!

💡 추천 사용 환경: 교실 전자칠판이나 태블릿 (브라우저만 있으면 어디서든 OK). 청소 모둠 인원수에 맞춰 역할 개수를 정해주세요.

🚀 시작하는 방법

  1. 전자칠판 브라우저(크롬/엣지)로 사이트 열기 위 주소를 주소창에 입력하세요. 처음 들어가면 예시 역할 5개(쓸기·대걸레·환기 등)가 자동으로 보여요.
  2. 커버 화면에서 우리 반 이름 입력 "3학년 4반" 같이 우리 반 이름을 적고 ✨ 시작하기! 버튼을 눌러요. 한 번 입력하면 다음에도 자동으로 기억돼요.
  3. 🎯 역할 설정에서 우리 반 청소에 맞게 수정 예시 역할은 ✏️ 수정하거나 🗑️ 삭제하고, ＋ 역할 추가하기로 새 역할을 만들어요. 이모지·이름·설명·카드 색상까지 자유롭게!
  4. "역할 뽑기! →" 눌러서 카드 뽑기 화면으로 학생들이 한 명씩 나와 카드를 탭하면 자기 역할이 짜잔 공개돼요. 모든 카드가 뒤집히면 🧹 청소 합시다! 버튼이 나타나요.
  5. 청소 끝낸 사람은 자기 카드를 탭해서 완료 표시 다 끝나면 자동으로 🎉 완료 화면으로 넘어가요. 폭죽이 터지고 칭찬 메시지로 마무리돼요!

🎬 화면 흐름 한눈에 보기

STEP 1
📝 커버
우리 반 이름 입력 → 시작하기
STEP 2
🎯 역할 설정
청소 역할 추가·수정·삭제
STEP 3
🃏 카드 뽑기
학생이 카드 탭 → 랜덤 역할 공개
STEP 4
🧹 청소 중
끝낸 사람이 자기 카드 탭하면 ✅
STEP 5
🎉 청소 완료!
폭죽 + 칭찬 메시지로 멋지게 마무리. "다시 시작" 누르면 역할 설정 화면으로 돌아가요.

🖥️ 전자칠판 바탕화면에 바로가기 등록하기

매번 주소를 입력하지 않아도, 바탕화면 아이콘 하나로 바로 열 수 있어요!

방법 1. 크롬(Chrome)인 경우

  1. 크롬으로 사이트 접속 주소창에 사이트 주소를 입력해서 들어가세요.
  2. 오른쪽 위 점 세 개(⋮) 메뉴 클릭 브라우저 우측 상단에 있는 메뉴 버튼이에요.
  3. '전송 및 저장' → '바로가기 만들기' 클릭 크롬 버전에 따라 '도구 더보기' → '바로가기 만들기'일 수도 있어요.
  4. 이름을 '청소 역할 뽑기'로 바꾸고 '만들기' 클릭 바탕화면에 바로가기 아이콘이 생겨요! 더블클릭하면 바로 열려요.

방법 2. 엣지(Edge)인 경우

  1. 엣지로 사이트 접속 주소창에 사이트 주소를 입력해서 들어가세요.
  2. 오른쪽 위 점 세 개(⋯) 메뉴 클릭 브라우저 우측 상단에 있는 메뉴 버튼이에요.
  3. '앱' → '이 사이트를 앱으로 설치' 클릭 팝업이 나오면 '설치'를 눌러주세요.
  4. 바탕화면에 아이콘이 자동 생성! 앱처럼 깔끔하게 열려요. 주소창 없이 전체 화면으로 쓸 수 있어서 전자칠판에 더 좋아요.
💡 엣지 앱 설치를 추천해요! 주소창이나 탭 없이 앱처럼 열려서 전자칠판 화면을 더 넓게 쓸 수 있어요. 카드 뽑기 연출도 훨씬 시원시원하게 보여요.

청소 끝난 사람이 카드 탭 → 자동 완료 표시

🔄

예시 역할 5개로 한 번에 되돌리기

💾 데이터 저장에 대해

이 도구는 각 전자칠판 안에만 역할 설정이 저장돼요. 인터넷 서버에 올라가지 않아서 학급별로 완전히 독립적으로 운영되고, 한 번 만들어둔 역할은 다음에 들어가도 그대로 남아 있어요.

⚠️ 꼭 알아두실 점

  • 같은 컴퓨터에서 브라우저를 바꾸면 역할 설정이 따로 저장돼요. 항상 같은 브라우저(예: 크롬)로 접속해 주세요.
  • 시크릿/InPrivate 모드는 사용 금지! 창을 닫으면 우리 반에 맞게 만든 역할이 모두 사라져요.
  • 브라우저 '사이트 데이터 삭제'를 누르면 역할 설정과 반 이름이 다 날아가요. 청소 작업할 때 주의하세요.
  • 청소 '진행률 체크'는 저장되지 않아요. 매 청소 시간마다 새로 시작되는 일회용이라, 부담 없이 매일 사용하시면 돼요.
💡 모둠별 운영 팁: 청소 모둠이 매주 바뀐다면, 모둠원 수에 맞춰 역할 개수만 조절하면 돼요. 예시 역할 5개는 5명 기준이고, ＋ 역할 추가하기로 늘리거나 🗑️ 삭제로 줄일 수 있어요.

💡 활용 팁

🎯 학기 시작 시
우리 반 교실 구조에 맞는 역할들을 미리 등록해두세요. 학생들과 함께 "어떤 역할이 있으면 좋을까?" 회의로 정해도 좋아요.
🃏 매일 청소 전
청소 모둠 친구들이 한 명씩 나와서 카드를 뽑게 하세요. "오늘 누가 뭐 해?"의 잔소리가 사라지고, 아이들이 게임처럼 즐거워해요.
✅ 청소 중
자기 역할이 끝난 학생은 자기 카드를 탭하게 해주세요. 진행률이 한눈에 보여서 서로 도와주는 분위기가 만들어져요.
🎉 청소 마무리
완료 화면의 폭죽 연출이 작은 보상이 돼요. "오늘도 수고했어요!" 한마디와 함께 마치면 청소 시간이 더 따뜻해져요.

❓ 자주 묻는 질문

우리 반은 청소 모둠이 6명인데, 역할도 6개로 만들 수 있나요?
네! ＋ 역할 추가하기 버튼으로 원하는 만큼 늘릴 수 있어요. 반대로 4명 모둠이면 🗑️ 삭제로 4개로 줄여도 돼요. (단, 최소 2개는 남겨야 해요.)
한 번 만든 역할 설정은 다음에도 그대로 남아 있나요?
네, 같은 컴퓨터·같은 브라우저로 접속하면 그대로 남아 있어요. 매일 다시 만들 필요 없이 한 번만 우리 반에 맞게 등록해두면 돼요.
청소 중에 실수로 카드를 탭했어요. 취소할 수 있나요?
청소 진행 화면에서는 한 번 누르면 완료로 표시돼요. 필요하다면 🎊 청소 끝! 버튼을 눌러서 종료한 뒤 "다시 시작"으로 새로 뽑으면 깔끔해요.
역할 카드 색상이나 이모지가 마음에 안 들어요!
언제든 ✏️ 수정 버튼을 누르면 이모지·이름·설명·색상을 모두 바꿀 수 있어요. 이모지 피커에서 클릭만 해도 바로 적용돼요.
예시 역할로 다시 돌아가고 싶어요.
역할 설정 화면 아래쪽 🔄 기본값으로 버튼을 누르면 돼요. 한 번 더 누르면 확정되니, 실수 방지용으로 두 번 클릭 구조예요.
스마트폰으로도 사용 가능한가요?
가능해요. 반응형이라 모바일 화면에도 맞춰져요. 다만 학급 전체가 함께 보는 도구라 큰 화면(전자칠판/태블릿)을 권장해요.
