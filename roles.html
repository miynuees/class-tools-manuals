1인1역 다이어리

매일의 작은 역할, 1년의 큰 성장 👥

https://miynuees.github.io/class-roles-tool/

🌟 어떤 도구인가요?

학급 1인1역을 매일 체크하고, 1년간의 활동을 기록·확인하는 학급 운영 도구예요. 우리 반의 역할과 학생들을 자유롭게 등록하고, 매일 누가 어떤 역할을 했는지 한 번의 클릭으로 기록할 수 있어요.

연말에는 역할별·학생별로 1년 동안의 활동을 한눈에 볼 수 있어서, 생활기록부 작성이나 학생 면담에도 활용하기 좋아요!

💡 추천 사용 환경: 교실 전자칠판이나 태블릿 (브라우저만 있으면 어디서든 OK)

🚀 시작하는 방법

  1. 전자칠판 브라우저(크롬/엣지)로 사이트 열기 위에 있는 주소를 주소창에 입력하세요. 처음 들어가면 예시 역할 5개가 자동으로 보여요.
  2. 제목 클릭해서 우리 반 이름 바꾸기 제목 옆 ✏️ 아이콘이나 제목 자체를 누르면 반 이름을 자유롭게 바꿀 수 있어요. (예: "3학년 1반 1인1역")
  3. ⚙️ 편집 메뉴에서 우리 반 역할과 학생 등록 예시 역할은 삭제하거나 수정하고, 우리 반에 맞는 역할과 학생들을 등록하세요. 이모지도 클릭해서 자유롭게 변경 가능!
  4. 매일 종례 시간에 역할 카드를 누르고 한 학생 체크 역할 카드를 누르면 학생 목록이 떠요. 오늘 그 역할을 한 학생을 선택하면 자동으로 진행률이 채워져요.
  5. 📅 연간기록에서 한 해 활동 한눈에 확인 역할별 12개월 달력과 학생별 누적 기록을 볼 수 있어요. 학생별 카드를 누르면 그 학생의 개인 활동 달력도 볼 수 있어요!

🖥️ 전자칠판에 앱으로 설치하기

매번 주소를 입력하지 않아도, 바탕화면 아이콘 하나로 앱처럼 바로 열 수 있어요!

방법 1. 크롬(Chrome)에서 설치하기 ⭐ 추천

  1. 크롬으로 사이트 접속 주소창에 위 사이트 주소를 입력해서 들어가세요.
  2. 주소창 오른쪽 끝의 설치 아이콘(💻 또는 ⊕) 클릭 주소창 안쪽 오른쪽 끝에 작은 설치 아이콘이 떠 있어요. 안 보이면 메뉴(⋮) → '캐스트, 저장 및 공유' → '페이지를 앱으로 설치' 를 눌러주세요. (크롬 버전에 따라 메뉴 위치가 살짝 달라요.)
  3. 팝업이 뜨면 '설치' 클릭 바탕화면과 시작 메뉴에 앱 아이콘이 자동으로 생겨요!
  4. 앱 아이콘 더블클릭으로 실행 주소창·탭 없이 앱처럼 열려서 전자칠판 화면을 더 넓게 쓸 수 있고, 실수로 다른 페이지로 이동할 일도 없어요.

방법 2. 엣지(Edge)에서 설치하기

  1. 엣지로 사이트 접속 주소창에 사이트 주소를 입력해서 들어가세요.
  2. 오른쪽 위 메뉴(⋯) → '앱' → '이 사이트를 앱으로 설치' 클릭 팝업이 뜨면 '설치'를 눌러주세요. 바탕화면에 앱 아이콘이 자동으로 생기고, 크롬과 똑같이 앱처럼 열려요.
💡 크롬을 추천해요! 대부분의 선생님들이 쓰시는 브라우저라 가장 편하고, 한 번 설치해두면 매일 클릭 한 번으로 바로 열 수 있어요. 엣지를 쓰셔도 동일하게 앱 설치가 가능해요.

💾 데이터 저장에 대해

이 도구는 각 전자칠판 안에만 데이터가 저장돼요. 인터넷 서버에 올라가지 않아서 학생 이름 등 개인정보가 외부로 새어나갈 걱정이 없고, 학급별로 완전히 독립적으로 운영돼요.

⚠️ 꼭 알아두실 점

  • 같은 컴퓨터에서 브라우저를 바꾸면 데이터가 따로 저장돼요. 항상 같은 브라우저(예: 크롬)로 접속해 주세요.
  • 시크릿/InPrivate 모드는 사용 금지! 창을 닫으면 모든 데이터가 사라져요.
  • 브라우저 '사이트 데이터 삭제'를 누르면 1년간의 기록이 모두 날아가요. 청소 작업할 때 주의하세요.
  • 학기 중간중간 백업을 권장해요. (💾 백업 → 데이터 내보내기)
💾 백업하는 방법: 헤더 우측 💾 백업 버튼 → '데이터 내보내기' 클릭 → JSON 파일이 다운로드돼요. USB나 클라우드에 보관해두면 컴퓨터를 바꿔도 안전하게 옮길 수 있어요. 복원도 같은 메뉴에서 '데이터 불러오기'로 가능해요.

💡 활용 팁

🎯 학기 시작 시
우리 반에 필요한 역할들을 미리 등록해두세요. 학생들과 함께 어떤 역할이 있으면 좋을지 회의로 정해도 좋아요!
📅 매일 사용
조회나 종례 시간에 그날의 역할 체크를 함께 해보세요. 학생이 직접 자기 활동을 클릭하게 하면 책임감도 길러져요.
📊 학기말 / 연말
연간기록 탭에서 학생별로 1년간 어떤 활동을 했는지 한눈에 확인! 생활기록부 작성이나 칭찬 카드 만들 때 활용하세요.
💾 정기 백업
학기 중간(중간고사 즈음)과 학기말에 한 번씩 백업을 해두면, 컴퓨터 문제가 생겨도 안심이에요.

❓ 자주 묻는 질문

다른 학교나 다른 반의 데이터와 섞이지 않나요?
절대 섞이지 않아요. 데이터는 각 컴퓨터 브라우저 안에만 저장되어서, 우리 반 전자칠판은 우리 반 데이터만 가지고 있어요.
학생 이름을 중간에 바꾸면 어떻게 되나요?
기존 활동 기록의 이름은 자동으로 바뀌지 않아요. 그래서 학기 초에 이름을 정확하게 등록해두시는 걸 권장해요.
우리 반에 같은 이름의 학생이 있어요!
같은 이름은 한 명으로 합산돼요. 구분이 필요하면 "수민A", "수민B" 같이 등록하시면 돼요.
실수로 체크했는데 취소할 수 있나요?
네! 역할 카드를 다시 누르고 같은 학생을 한 번 더 클릭하면 체크가 취소돼요.
컴퓨터를 새로 바꾸면 기존 기록을 옮길 수 있나요?
가능해요! 기존 컴퓨터에서 💾 백업 → 데이터 내보내기로 JSON 파일을 저장한 뒤, 새 컴퓨터에서 데이터 불러오기로 복원하면 돼요.
스마트폰으로도 사용 가능한가요?
가능해요. 반응형이라 모바일 화면에 맞게 카드 크기가 자동 조정돼요. 다만 학급 활동용이라 큰 화면(전자칠판/태블릿)을 권장해요.
